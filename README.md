# README

Ruby version 2.5.1,

Télécharger le repo, unzip.

Place-toi dans le terminal à la racine du dossier et entre les commandes suivantes :
$bundle install
$yarn install --check-files
$rails db:migrate
$rails db:seed
$rails s -p 3000

Welcome page adresse : http://localhost:3000/welcome/"ton_prénom"
